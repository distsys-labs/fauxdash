# fauxdash

This repository needs to adapt to several considerations:

* This is an npm module that should be usable in Node versions 18 or greater
* The source code is written in TypeScript and should be usable from both TypeScript and JavaScript
* Tests should be written using vitest
* Use semistandard for linting with eslint plugins for typescript
* Include a Makefile with commands for build, test and release
* Use standard-version to cut new releasess
